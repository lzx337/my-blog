📖 Markdown 完整样式指南

2026年5月24日 约 8 分钟 264 次浏览
分类：指南|标签：Markdown样式指南排版

本指南展示了本站 Markdown 渲染后的最终样式，包括标题、列表、代码块、表格、引用等所有常用元素。你可以直接对照编写内容，确保排版美观统一。

六级标题 (H6) — 极细分类

✍️ 文本样式

这是普通段落文本，字体圆润，行高舒适。你可以使用 粗体强调、斜体语气 或者 粗斜体混合。另外支持 删除线 和 高亮标记。内联代码示例：git commit -m "feat: style guide"，适合技术文案。

💻 代码块与语法高亮

支持多种编程语言，通过 ``` 标记并声明语言类型，将呈现美观背景与等宽字体。

// 一个轻量级防抖函数示例
function debounce(fn, delay) {
  let timer = null;
  return function(...args) {
    clearTimeout(timer);
    timer = setTimeout(() => fn.apply(this, args), delay);
  };
}

window.addEventListener('resize', debounce(() => {
  console.log('窗口大小已调整');
}, 300));

💬 引用块

“代码是写给人看的，只是恰好能被机器执行。” — 计算机领域格言。引用块使用 > 标记，带有左侧边框和柔和背景色。

可以多段引用，并且嵌套其他元素如列表或代码块。

💡 提示： 此类自定义容器同样可用，通过组合边框和背景色营造提示、警告、危险等区块。后续可扩展为 Admonitions 样式。

🖼️ 图片与链接

图片默认圆角、自适应宽度并带细微阴影。也可以使用外部链接。

更多语法参考：Markdown Guide 官网，点击外部链接带有安全提示。

📎 脚注与水平分割线

Markdown 脚注示例：这里有一段需要注解的内容[1]，可以指向页面底部释义。

水平分割线以上使用 --- 或 *** 创建。

🧮 数学公式 (扩展支持)

若需展示数学公式，可使用行内公式：$E = mc^2$ 渲染为 $E = mc^2$，块级公式：$$ \int_{a}^{b} f(x) \, dx = F(b) - F(a) $$（本站通过 MathJax 或 Katex 支持，本示例仅为样式占位）
