# 个人博客网站 - 本地运行指南

## 📋 项目概述

这是一个具有高级视觉效果和多功能特性的专业个人博客网站，由六人团队历时三个月精心打造。

### ✨ 核心特性

- **动态背景系统**: 基于时间变化的智能背景颜色切换（8个时间节点）
- **8个功能页面**: 首页、文章列表、文章详情、关于作者、作品集、联系方式、归档、搜索
- **高级视觉效果**: 视差滚动、粒子动画、渐变过渡、微交互反馈、悬停特效
- **纯前端架构**: 无需后端服务器，双击HTML即可运行
- **JSON驱动内容管理**: 通过修改JSON文件轻松更新内容
- **响应式设计**: 适配桌面端、平板及移动设备

---

## 🚀 快速启动

### 方法一：双击运行（推荐）

1. 直接双击 `index.html` 文件
2. 浏览器将自动打开网站

### 方法二：使用本地服务器

```bash
# Python 3
python -m http.server 8000

# Node.js (npx)
npx serve .

# PHP
php -S localhost:8000
```

然后访问 http://localhost:8000

## 🎨 内容管理

### 更新文章

编辑 `data/articles.json` 文件：

```json
{
  "id": 9,
  "title": "新文章标题",
  "slug": "new-article-slug",
  "excerpt": "文章摘要",
  "content": "## 文章内容（支持Markdown）",
  "category": "技术",
  "tags": ["标签1", "标签2"],
  "author": "作者名",
  "date": "2026-03-20",
  "readTime": "5 分钟",
  "featured": true,
  "views": 0,
  "likes": 0
}
```

### 更新项目

编辑 `data/projects.json` 文件：

```json
{
  "id": 7,
  "title": "新项目名称",
  "slug": "new-project",
  "description": "项目描述",
  "category": "Web应用",
  "tags": ["React", "Node.js"],
  "thumbnail": "project.jpg",
  "link": "https://example.com",
  "github": "https://github.com/example",
  "date": "2026-03",
  "featured": true,
  "technologies": ["React", "Node.js", "MongoDB"]
}
```

### 更新个人信息

编辑 `data/profile.json` 文件，包括：
- 基本信息（姓名、头像、个人简介）
- 社交媒体链接
- 技能列表
- 工作经历
- 教育背景
- 团队成员

## 🎯 功能使用

### 导航

- 点击顶部导航链接切换页面
- 支持键盘导航（Tab 键）
- URL hash 路由支持（刷新保持状态）

### 主题切换

- 点击右上角 🌙/☀️ 按钮切换深色/浅色模式
- 设置自动保存到本地

### 字体大小

- 点击 A⁺ 按钮调整字体大小
- 三档可选：小、中、大

### 文章搜索

- 点击搜索图标或使用快捷键 `Ctrl/Cmd + K`
- 支持标题、内容和标签搜索
- 实时显示搜索结果

### 收藏文章

- 在文章详情页点击 ❤️ 按钮收藏
- 收藏列表显示在侧边栏

### 评论系统

- 在文章详情页底部发表评论
- 评论数据保存在本地 localStorage

## 📱 响应式断点

| 设备 | 宽度 | 布局调整 |
|------|------|----------|
| 超大屏 | > 1280px | 完整多列布局 |
| 桌面 | 1024px - 1280px | 双列/单列切换 |
| 平板 | 768px - 1024px | 单列布局 |
| 大手机 | 480px - 768px | 紧凑布局 |
| 手机 | < 480px | 最小化布局 |
